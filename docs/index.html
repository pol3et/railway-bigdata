04

The final dataset · Gold feature matrix

The deliverable is a wide feature matrix (not StatFact): one row per (geo, year), one column per canonical feature. StatFact is the Silver shape — a long table, one row per observation, carrying source_system. Gold is built by pivoting StatFact long→wide (gold/build.py · pivot_stats); when the same (geo, year, feature) comes from more than one source, a priority policy (eurostat > uic > ksh > statistik_austria > worldbank) keeps one value, and the source_system column is then dropped — so Gold itself stores no source column. The source tag below is reconstructed from Silver provenance, for illustration only.
